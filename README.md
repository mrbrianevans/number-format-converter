# Number format converter
Open source project to convert numbers between different formats.

## Tech summary
**Backend:** nodeJS , expressJS

**Frontent:** ReactJS , Bootstrap

## Formats
- binary
- octal
- decimal
- hexadecimal
- base 12
- base 64
- roman numerals
- english words
